QuasarDB datasource for Grafana
===============================


Installing (for now)
--------------------

Requires [node](https://nodejs.org/en/) and [yarn](https://yarnpkg.com/en/).

```sh
cd <grafana-root>/data/plugins
git clone git@github.com:bureau14/qdb-grafana-plugin.git
yarn && yarn build
```

Now start Grafana.
